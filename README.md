# Kuyen

> **Kuyen** (Mapudungun for *Moon*): A next-generation, premium lunar-themed platform built for high-end digital experiences.

## ✨ Features

- **Premium Aesthetic**: Dark-themed UI with glassmorphism and fluid typography.
- **Vite Powered**: Ultra-fast development and build cycles.
- **Modern Stack**: Vanilla JS and CSS for maximum control and performance.
- **Responsive**: Fully optimized for all screen sizes.

## 🚀 Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) (Latest LTS recommended)
- [npm](https://www.npmjs.com/)

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/yordyy-eng/kuyen.git
   ```

2. Navigate to the project directory:
   ```bash
   cd kuyen
   ```

3. Install dependencies:
   ```bash
   npm install
   ```

### Development

Start the development server:
```bash
npm run dev
```

### Build

Create a production-ready bundle:
```bash
npm run build
```
